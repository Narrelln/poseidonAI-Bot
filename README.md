
# 🤖 Poseidon AI Dashboard

Poseidon is an advanced AI-powered crypto trading dashboard designed for **meme coin and futures trading** with real-time wallet tracking, volume analysis, bonding insights, and sniper wallet detection.

---

## 🚀 Features

- 🔍 **Wallet Feed**: Live tracking of smart wallet activity via Helius API
- 📈 **Bonding & Volume Tracker**: Displays bonding % and 24h volume from Birdeye
- 🎯 **Trade Execution Engine**: Filters tokens based on bonding %, volume, market cap, and smart wallet convergence
- 💡 **Entry Labels**: Identifies if a token qualifies for “Early Entry” (MC < $10K) or “Bonded Entry”
- 🌐 Fully browser-based UI with no backend setup required

---

## 📂 Files Included

- `index.html` – main dashboard layout
- `style.css` – responsive neon-styled theme
- `execution.js` – core logic for trade filtering and execution
- `helius.js` – wallet feed integration
- `birdeye.js` – bonding & volume data fetch

---

## 🛠️ How to Use

1. Clone the repo or download the zip
2. Launch `index.html` using **Live Server** (VS Code extension recommended)
3. Customize token feeds or connect real-time mints
4. Use GitHub Pages or Vercel to host it live

---

## 📡 Live Deployment

> After pushing your files to GitHub:

- Go to **Settings → Pages**
- Select **Deploy from branch**
- Choose `/ (root)` from `main`
- ✅ Your dashboard will be live at:
  https://<your-username>.github.io/poseidon-dashboard/
